<h2 class="card-header" id="gwc-events-booking-policy">GWC Events Booking Policy</h2>

<p>To keep the club running and ensure that we meet all our obligations as a Ministry of Culture organisation, all members who wish to use the club for events are required to follow this process and to register their proposed event with the Gibraltar Warhammer Club committee.</p>

<h2 class="card-header" id="registering-an-event">Registering an Event</h2>

<p>Before you go ahead with any event at the club, you must register your proposal with the committee using the <a href="#event-registration-form" class="link">form below</a>. This lets us check that the date is available, understand what you are planning, and make sure the space is prepared for you.</p>

<p>The committee will confirm availability and let you know if anything else is needed before your event goes ahead.</p>

<h2 class="card-header" id="types-of-event">Types of Event</h2>

<h3 class="policy-subheader" id="member-events">Member Events</h3>

<p>Member events are always free to organise, provided all those attending are a GWC member. In special circumstances, plus-ones can be allowed if agreed with the GWC committee.</p>

<p>Make sure to register your members event using the <a href="#event-registration-form" class="link">form below</a> so that we can confirm the date is available and note any special requirements, so the club can be prepared. As usual, members are expected to follow the <a href="/code-of-conduct/" class="link">club rules</a> and to leave the space as they found it.</p>

<h3 class="policy-subheader" id="private-events">Private Events</h3>

<p>Private events are bookings where you take exclusive use of the club for a closed group. This might be a birthday party, a workplace team day, a private tournament for invited guests, or a similar occasion.</p>

<p>Private hire is charged at <strong>£75 per four-hour block</strong> of venue use. If you need the space for longer, each additional four-hour block is charged at the same rate. All regular club rules apply during the event, including that no children under 14 may be in attendance without the presence of a parent or guardian.</p>

<p>Depending on the nature of the event, the committee may require a deposit before the booking is confirmed. The event organiser is liable for any damage caused during the event.</p>

<p>Please register your proposal using the <a href="#event-registration-form" class="link">form below</a> so we can confirm dates, fees, and any conditions before you confirm your booking.</p>

<h3 class="policy-subheader" id="public-events">Public Events</h3>

<p>Because the Gibraltar Warhammer Club is a Ministry of Culture organisation, all public events must be properly organised with the GWC committee (or with Dice Bastion / SN Battle Reports acting as the committee's representatives). This ensures that we meet the club's legal and organisational requirements, stick to our policies, and follow the proper procedures.</p>

<p>To propose a public event, please use the <a href="#event-registration-form" class="link">form below</a>, giving as much detail as possible.</p>

<p>The committee will review your proposal and may ask for further information, set conditions, or suggest changes before giving approval. We aim to support member-led events where we can, but we may decline proposals that do not fit the club's calendar or obligations.</p>

<p>Approved public events may be subject to additional requirements, such as advance notice for publicity, agreed start and finish times, and a clear plan for setup and cleanup.</p>

<h2 class="card-header" id="general-expectations">General Expectations</h2>

<p>Whatever type of event you are running, the following always apply:</p>

<ul class="policy-list">
<li><span class="bullet"></span><span>Follow the <a href="/code-of-conduct/" class="link">club code of conduct</a> and treat the venue, equipment, and other users with respect.</span></li>
<li><span class="bullet"></span><span>Leave the space tidy and return furniture and club property to where you found it.</span></li>
<li><span class="bullet"></span><span>Do not move club games, terrain, tools, or equipment without prior permission from the committee.</span></li>
<li><span class="bullet"></span><span>Report any damage, safety concerns, or incidents to the committee immediately.</span></li>
</ul>

<script>
(function () {
const EVENTS_API = `${window.utils.getApiBase(true)}/events/register`;
const TS_SITE_KEY = '0x4AAAAAACAB4xlOnW3S8K0k';
const IS_LOCALHOST = ['localhost', '127.0.0.1', '0.0.0.0'].includes(window.location.hostname);

const form = document.getElementById('event-registration-form-el');
const errorEl = document.getElementById('event-reg-error');
const submitBtn = document.getElementById('event-reg-submit');
const formState = document.getElementById('event-form-state');
const successState = document.getElementById('event-form-success');
const turnstileState = { widgetId: null };

function showError(msg) {
if (!errorEl) return;
errorEl.textContent = msg || 'Something went wrong. Please try again.';
errorEl.style.display = msg ? 'block' : 'none';
}

function prefillFromSession() {
const user = utils.session.getUser();
if (!user) return;
const nameEl = document.getElementById('event-reg-name');
const emailEl = document.getElementById('event-reg-email');
if (nameEl && user.name && !nameEl.value) nameEl.value = user.name;
if (emailEl && user.email && !emailEl.value) emailEl.value = user.email;
}

async function initTurnstile() {
try {
await window.utils.renderTurnstile('event-reg-ts', TS_SITE_KEY, {
skipOnLocalhost: IS_LOCALHOST,
widgetState: turnstileState,
appearance: 'interaction-only'
});
} catch (e) {
console.warn('Turnstile failed to load:', e);
}
}

async function getTurnstileTokenWithWait() {
const start = Date.now();
while (Date.now() - start < 8000) {
try {
return await window.utils.getTurnstileToken('event-reg-ts', turnstileState.widgetId, IS_LOCALHOST);
} catch (err) {
if (!String(err?.message || '').toLowerCase().includes('security check')) throw err;
await new Promise((resolve) => setTimeout(resolve, 300));
}
}
throw new Error('Please complete the security check.');
}

if (form) {
form.addEventListener('submit', async function (e) {
e.preventDefault();
showError('');

if (document.getElementById('event-reg-hp').value) return;

const name = document.getElementById('event-reg-name').value.trim();
const email = document.getElementById('event-reg-email').value.trim();
const phone = document.getElementById('event-reg-phone').value.trim();
const eventType = document.getElementById('event-reg-type').value;
const eventDetails = document.getElementById('event-reg-details').value.trim();

if (!name) { showError('Please enter your name.'); return; }
if (!email || !/^\S+@\S+\.\S+$/.test(email)) { showError('Please enter a valid email address.'); return; }
if (!phone || phone.replace(/\D/g, '').length < 6) { showError('Please enter a valid phone number.'); return; }
if (!eventType) { showError('Please select an event type.'); return; }
if (eventDetails.length < 20) {
showError('Please provide more detail about your proposed event (at least 20 characters).');
return;
}

let turnstileToken;
try {
turnstileToken = await getTurnstileTokenWithWait();
} catch (err) {
showError(err.message || 'Security check failed. Please refresh and try again.');
return;
}

submitBtn.disabled = true;
submitBtn.textContent = 'Sending…';

try {
const res = await fetch(EVENTS_API, {
method: 'POST',
headers: { 'Content-Type': 'application/json' },
body: JSON.stringify({ name, email, phone, eventType, eventDetails, turnstileToken })
});
const data = await res.json();

if (!res.ok || !data.ok) {
const msgs = {
name_required: 'Please enter your name.',
invalid_email: 'Please enter a valid email address.',
invalid_phone: 'Please enter a valid phone number.',
invalid_event_type: 'Please select an event type.',
details_too_short: 'Please provide more detail about your proposed event (at least 20 characters).',
turnstile_failed: 'Security check failed. Please refresh and try again.',
rate_limit_exceeded: data.message || 'Too many submissions. Please wait a few minutes and try again.',
send_failed: data.message || 'Could not send your registration. Please try again later.'
};
showError(msgs[data.error] || data.message || 'Something went wrong. Please try again.');
submitBtn.disabled = false;
submitBtn.textContent = 'Submit registration';
return;
}

if (formState) formState.style.display = 'none';
if (successState) successState.style.display = 'block';
} catch (err) {
showError('Network error. Please check your connection and try again.');
submitBtn.disabled = false;
submitBtn.textContent = 'Submit registration';
}
});
}

prefillFromSession();
initTurnstile();
})();
</script>
