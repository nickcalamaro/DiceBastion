Newsletter API Test

Auth

Open the admin panel, log in, then run localStorage.getItem('sessionToken') in the console to get your token.

GET /admin/newsletters — List all newsletters

POST /admin/newsletters — Create draft

GET /admin/newsletters/:id — Fetch single

PUT /admin/newsletters/:id — Update

POST /admin/newsletters/:id/send — Send immediately

This sends to ALL opted-in recipients for real. Use a test draft.

DELETE /admin/newsletters/:id — Delete

GET /admin/newsletter/recipients — Recipient count
